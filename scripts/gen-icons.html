AirPair — icônes PNG

Ouvre ce fichier dans un navigateur, puis clique sur les liens pour télécharger les PNG.
Place les fichiers dans src/public/ et ajoute-les au manifest.

Après téléchargement, ajouter dans src/public/manifest.webmanifest :

{ "src": "icon-192.png", "sizes": "192x192", "type": "image/png" }
{ "src": "icon-512-maskable.png", "sizes": "512x512", "type": "image/png", "purpose": "maskable" }
